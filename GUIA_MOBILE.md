# 📱 Guia de Instalação Mobile (Bulldog Frida Arcade)

Para jogar como um aplicativo real no seu celular, siga os passos abaixo:

## No Android (Google Chrome)
1. Abra o navegador **Chrome** no seu celular.
2. Acesse o link do seu jogo (ex: `https://piresdba.github.io/Bulldog-Frida-Arcade/`).
3. Aguarde alguns segundos. Aparecerá um banner escrito "**Instalar na tela inicial**".
4. Toque em instalar e confirme. O ícone da **Frida** (Bulldog) surgirá na sua lista de aplicativos!

## No iOS (Safari - iPhone/iPad)
1. Abra o **Safari** e acesse o link do seu jogo.
2. Toque no botão **Compartilhar** (ícone do quadrado com seta para cima na barra inferior).
3. Role a lista para baixo e toque em **"Adicionar à Tela de Início"**.
4. Toque em "Adicionar" no canto superior direito.

---

Este projeto foi otimizado para celulares (PWA - Progressive Web App). 
Ele funciona com controles touch nativos e orienta automaticamente a tela para o modo paisagem (horizontal).
